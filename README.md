### Установка

1) Скачать и положить в папку со шрифтами
2) Подключить:

```bash
#include "gfx_fonts/AbtSmall8pt7b.h"
DMD_GFX_Font AbtSmall((uint8_t*)&AbtSmall8pt7b, (uint8_t*)&AbtSmall8pt8b_rus, 0x80, 9);
```
